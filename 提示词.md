修改 `backend/achievements/models.py`：

   - 关系 (Edges)：
     - `(Teacher)-[:PUBLISHED {is_first: true/false}]->(Paper)`
     - `(ExternalScholar)-[:CO_AUTHORED]->(Paper)`
     - `(Paper)-[:HAS_KEYWORD]->(Keyword)`

## 步骤 4：前端 Vue3 + ECharts 组件
在 `frontend/src/views/` (或 components) 下创建 `AcademicGraph.vue`：
1. 使用 Vue 3 `<script setup lang="ts">`。

4. 渲染一个关系图谱 (series type: 'graph')。
5. **样式要求：** 开启 `force` 力引导布局，节点支持拖拽，不同类型的节点（如中心教师、校内合作者、校外学者、关键词）需要用不同的颜色和 `symbolSize` 区分，并在 `tooltip` 中显示详细信息。

# Constraints (约束)

- 所有的 Python 代码必须包含清晰的类型提示 (Type Hints) 和中文注释。
- Neo4j 的连接配置请读取 Django `settings.py` 中的环境变量（给出配置示例）。
- 前端使用 Axios 发送请求，注意携带 Bearer Token。
