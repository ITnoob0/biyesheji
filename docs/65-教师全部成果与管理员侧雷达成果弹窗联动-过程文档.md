# 65-教师全部成果与管理员侧雷达成果弹窗联动-过程文档

- 教师画像页展示教师全量成果
- 排序规则：

  - 其余按时间从近到远
- 移除教师页 `支撑成果 / 推荐解释 / 问答说明`
- 新增管理员雷达预览弹窗
- 新增管理员全部成果预览弹窗
- 教师管理、学院看板中的相关按钮不再跳页，改为弹窗查看
