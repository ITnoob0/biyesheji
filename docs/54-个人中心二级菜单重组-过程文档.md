# 54-个人中心二级菜单重组-过程文档

## 本次范围

- 仅调整前端“个人中心”一级菜单。
- 不改后端接口、字段和逻辑。
- 不调整画像主页、成果中心、推荐、问答等其他一级菜单。

## 新的二级菜单结构

- 公开资料
- 账户安全
- 快捷入口 / 代表成果

## 路由调整

- `/profile-editor` 现在直接落到“公开资料”。
- 原 `/profile-editor/public-profile` 保留兼容，重定向回 `/profile-editor`。
- 删除了菜单中的“总览”子项。

## 页面结构调整

### 公开资料

- 保留原个人资料主卡、资料预览和编辑表单。
- 去掉公开字段边界说明、字段标签说明和其他大段解释块。

### 账户安全

- 左侧：内部管理信息。
- 右侧：账户安全。
- 保留原有数据来源与密码修改逻辑。
- 去掉原先解释性 alert 块，改为卡片内简洁信息展示。

### 快捷入口 / 代表成果

- 左侧：代表成果速览。
- 右侧：统一成果。
- 复用原代表成果面板和快捷入口面板，不改数据来源。

## 删除的说明性文字块

- 个人中心原“总览”页的长说明结构。
- 公开资料页顶部/中部关于公开字段用途的 alert 说明。
- 内部管理信息关于内部字段用途的 alert 说明。
- 联系方式公开策略下的大段说明列表。
- 字段公开/内部边界说明卡片。
- 快捷入口面板顶部的说明性 alert。
- 账户安全面板顶部两条说明性 alert。
