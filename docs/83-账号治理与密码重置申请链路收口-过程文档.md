# 83-账号治理与密码重置申请链路收口-过程文档

## 1. 目标与范围
- 关闭所有自助注册能力，账号创建统一由管理员完成。
- 将“忘记密码”由自助改密改为“提交重置申请”流程。
- 新增两类管理员批量创建账号能力（Excel/CSV 导入）。
- 保持既有教师管理、通知中心、密码重置主链路兼容。

## 2. 后端改动

### 2.1 注册与忘记密码策略调整
- 文件：`backend/users/views.py`
  - `TeacherRegistrationView.post` 改为直接返回 `403`，错误码：`self_registration_disabled`。
  - `ForgotPasswordResetView.post` 改为“申请提交成功”语义，不再直接修改密码。

- 文件：`backend/users/serializers.py`
  - `ForgotPasswordResetSerializer` 入参改为：
    - `employee_id`
    - `real_name`
    - `department`
  - 校验工号 + 姓名 + 学院三元一致性。
  - 校验目标账号必须为教师账号（管理员账号禁止该入口）。
  - 查找同学院有效学院管理员并批量发送通知。

### 2.2 通知类型扩展
- 文件：`backend/users/models.py`
  - `UserNotification` 新增类别：
    - `CATEGORY_PASSWORD_RESET_REQUEST = "PASSWORD_RESET_REQUEST"`
  - 该通知用于学院管理员接收教师密码重置申请。

### 2.3 批量创建接口
- 文件：`backend/users/views.py`
  - 新增 `TeacherBulkImportView`，路径：
    - `POST /api/users/teachers/bulk-import/`
  - 角色边界：
    - 系统管理员：批量创建学院管理员（需导入学院列）。
    - 学院管理员：批量创建本学院教师（学院自动使用当前管理员学院）。
  - 固定规则：
    - 学院管理员批量创建教师：`title` 固定 `讲师`。
    - 系统管理员批量创建学院管理员：`title` 固定 `学院管理员`。
  - 默认密码：
    - 仍使用系统默认临时密码 `teacher123456`，并保留“需改密”标记。

- 文件：`backend/users/urls.py`
  - 新增路由：`teacher_bulk_import`。

### 2.4 Excel/CSV 解析能力
- 文件：`backend/users/services.py`
  - 新增 `parse_bulk_account_import_file`：
    - 支持 `.csv`、`.xlsx`
    - 自动识别首行表头（工号/姓名/学院等）
    - 支持无表头按列位回退读取
  - 供 `TeacherBulkImportView` 复用。

## 3. 前端改动

### 3.1 登录页与注册页
- 文件：`frontend/src/views/LoginView.vue`
  - 删除“教师注册”按钮入口。
  - 将“忘记密码”改为密码框下方蓝色小字链接。

- 文件：`frontend/src/views/TeacherRegisterView.vue`
  - 改为“自助注册已关闭”提示页，仅保留返回登录按钮。

### 3.2 忘记密码页改为申请流
- 文件：`frontend/src/views/ForgotPasswordView.vue`
  - 表单改为 `工号 + 姓名 + 所属学院`。
  - 提交文案改为“提交重置申请”。
  - 去除“新密码/确认新密码”字段与自助改密逻辑。

### 3.3 教师管理页批量创建与通知筛选联动
- 文件：`frontend/src/views/TeacherManagementView.vue`
  - 在创建页头部新增“批量创建”按钮。
  - 新增批量导入弹窗（支持 `.xlsx/.csv` 上传）。
  - 新增导入结果展示（成功数、跳过数、默认临时密码）。
  - 支持读取路由 `keyword` 查询参数并自动筛选，配合通知点击后精准定位教师。

### 3.4 类型定义同步
- 文件：`frontend/src/types/users.ts`
  - 新增 `TeacherBulkImportResponse`。
  - `UserNotificationCategory` 新增 `PASSWORD_RESET_REQUEST`。

## 6. 结果总结
- 自助注册已关闭。
- 自助忘记密码已关闭，改为学院管理员审批/处理链路。
- 两类管理员均支持按角色边界批量导入创建账号。
