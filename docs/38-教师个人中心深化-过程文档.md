# 38-教师个人中心深化-过程文档

## 1. 本轮目标

在 P0 已完成、个人中心已替代基础档案页的前提下，继续把教师个人中心从“档案页升级版”收口成教师侧最核心的统一工作台入口。

本轮重点包括：

- 头像上传能力
- 联系方式展示策略
- 公开字段 / 内部字段边界继续明确
- 个人中心摘要卡片与状态展示
- 教师默认工作台入口与返回链路统一
- 资料、成果、画像、推荐、问答之间的联动增强

## 2. 扫描结论

本轮开始前，仓库已具备：

- 教师个人中心页 `TeacherProfileEditorView.vue`
- 快捷入口组件 `PersonalCenterQuickLinks.vue`
- 画像主页、成果页、推荐页、问答页和统一顶部导航
- `GET/PATCH /api/users/me/` 统一资料更新入口
- 头像 URL 字段 `avatar_url`

仍然存在的问题主要有：

- 头像只支持手填 URL，不支持本地上传
- 联系方式虽然有邮箱和电话字段，但缺少展示策略控制
- 个人中心还更像“资料编辑页 + 快捷入口”，工作台摘要感不够强
- 教师默认品牌返回和默认工作台入口仍偏向画像页，不够符合“个人中心是统一入口”的目标

## 3. 本轮实现

### 3.1 后端

继续复用 `CustomUser` 和当前用户统一更新链路，只做最小必要增强：

- 在 `users.CustomUser` 新增 `contact_visibility`
  - 当前支持：
    - `email_only`
    - `phone_only`
    - `both`
    - `internal_only`
- 在当前用户返回中新增：
  - `contact_visibility`
  - `contact_visibility_label`
  - `public_contact_channels`
- 新增头像上传接口：
  - `POST /api/users/me/avatar/`
  - 使用现有用户模型上的 `avatar_url` 回写上传结果
  - 不新增头像表，不重做用户资料模型
- 新增媒体配置：
  - `MEDIA_URL`
  - `MEDIA_ROOT`
  - 开发环境下静态暴露 `media/`

说明：

- 头像上传当前是轻量媒体文件能力，不是完整素材管理系统
- 联系方式展示策略当前主要控制系统内资料摘要、画像摘要和推荐输入摘要，不代表公网展示体系

### 3.2 前端个人中心深化

在现有个人中心页基础上，做了以下增强：

- 教师默认工作台入口切换到个人中心
  - 教师登录后默认进入 `/profile-editor`
  - 顶部品牌按钮对教师优先返回个人中心
  - 管理员仍保持原有管理 / 画像视角
- 增加头像上传面板
  - 支持上传 JPG / PNG / WEBP / GIF
  - 仍保留头像 URL 作为备用方式
- 增加联系方式展示策略
  - 仅公开邮箱
  - 仅公开电话
  - 公开邮箱和电话
  - 仅内部可见
- 增加个人中心状态卡片
  - 公开资料完整度
  - 联系方式展示策略
  - 账户安全状态
  - 近期成果状态
- 公开资料预览区改为实时反映当前编辑中的联系方式公开策略
- 内部管理信息区补充：
  - 联系方式展示策略
  - 内部联系方式总览
- 快捷入口区补充“个人中心已作为教师默认工作台入口”的说明

### 3.3 文档同步

统一现状文档同步更新了以下内容：

- 接口清单补充头像上传接口和联系方式策略字段
- 模块边界文档说明：
  - 教师默认工作台入口已优先收口到个人中心
  - 当前头像上传仍是轻量媒体文件能力
  - 当前字段可见性控制保持克制
- 数据口径文档说明：
  - 联系方式按教师策略进行系统内展示
  - 联系方式策略不等于公网可见性平台
- README 补充教师个人中心的工作台定位说明

## 4. 测试与验证

### 4.1 后端测试增强

- 当前用户可通过个人中心上传头像
- 当前用户资料更新可写回联系方式策略

### 4.2 前端基线增强

在 `frontend/scripts/verify-third-round-ui.mjs` 补充：

- 联系方式展示策略标签解析
- 公开联系方式摘要生成
- 仅内部可见时的边界提示

## 6. 最小人工验证步骤

1. 使用教师账号登录
2. 确认默认进入个人中心，而不是画像页
3. 点击顶部品牌按钮，确认仍返回个人中心
4. 在个人中心上传本地头像，确认头像即时更新
5. 调整联系方式展示策略，观察公开资料预览区是否实时变化
6. 保存后进入画像、推荐、问答相关页面，确认资料摘要和跳转链路正常
7. 点击个人中心快捷入口，分别进入成果、画像、推荐、问答页，再通过顶部品牌按钮返回个人中心

## 7. 本轮边界说明

- 本轮没有推翻现有教师资料模型
- 本轮没有新增复杂头像资源表或素材平台
- 本轮没有把管理员字段暴露到教师公开资料区
- 本轮联系方式策略主要用于系统内展示，不代表公网开放策略
- 本轮教师工作台入口虽然切到个人中心，但管理员默认工作台入口仍保持原有管理 / 画像视角
