## 一、主链路验收

- 工号登录可用
- `sessionStorage` 登录态可用
- 登录失效后统一跳转与回跳可用
- 教师资料编辑可用
- 成果新增 / 编辑 / 删除 / 搜索 / 筛选可用
- 教师画像展示可用
- 学术拓扑图展示可用
- Neo4j 异常时 MySQL 回退可用

## 二、第三轮增强能力验收

- 推荐页支持筛选、排序、标签分组和管理员教师视角切换
- 推荐结果支持“智能解读”入口
- 学院看板支持学院 / 年份 / 教师筛选
- 学院看板支持从排行钻取到画像和推荐页
- 学院看板支持导出当前排行 CSV
- 智能问答页可生成至少 3 类说明结果
- 图谱页展示轻量图分析亮点
